# 导购聊天真人乱问回归清单

## 使用方式

- 每次修改聊天主链路、意图网关、记忆继承或协议编排后，至少跑一遍本清单
- 自动化通过后，再按本清单做人工 smoke test
- 重点观察 4 类问题：错拒、错路由、错继承、错组件返回

## 客户盘点

1. 现在有哪些客户
2. 目前门店都有哪些客户
3. 我想看看现在的会员池
4. 先给我看看客户大盘

## 客户筛选

5. 帮我找今天该优先跟进但还没联系的高净值客户
6. 我想看高净值客户
7. 给我筛一下最近没联系的重点客户
8. 帮我找最近适合回访的客户

## 品类 / 库存

9. 现在有哪些品类
10. 有哪些外套
11. 有哪些连衣裙
12. 现在门店还有哪些适合通勤的商品

## 商品推荐

14. 帮我推荐几件上班穿的
15. 给我找几个适合通勤的商品
16. 推荐几件轻薄一点的款

## 指定客户推荐

18. 乔安禾最近适合推荐什么
19. 给乔安禾挑 3 件通勤款
20. 按乔知夏的风格推荐几件

## 连续追问 / 上下文继承
